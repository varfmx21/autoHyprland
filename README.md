# autoHyprland

## Installation
1. Upgrade your packages.

```shell
Sudo Pacman -Syu
```

2. Clone this repository and navigate to it.
```shell
git clone https://github.com/varfmx21/autoHyprland
cd autoHyprland
```

3. Give executions permission to the script.
```shell
chmod +x setup.sh
```

4. Execute the script and follow instructions.
```shell
./setup.sh
```
